Codename One — Java compilado para JavaScript — O mesmo shell fino (remote-shell) dos apps iOS/Android/desktop, transpilado para a web pelo build na nuvem do Codename One. Comunica-se com o servidor via WebSocket bidirecional (protocolo JSON delta), renderizando os ViewStates recebidos com a UI nativa do CN1. Demonstra o padrão Cube MVP com segurança RSA + AES-GCM, a partir de uma única base Java.
